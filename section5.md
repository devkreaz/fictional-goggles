# Розділ 5. User Stories у UML-нотації

Нижче наведено UML-діаграми випадків використання для системи.

```mermaid
%%{init: {'theme':'base'}}%%
usecaseDiagram
    actor Читач
    actor Бібліотекар
    actor Адміністратор

    Читач --> (Реєстрація)
    Читач --> (Пошук книги)
    Читач --> (Бронювання книги)
    Читач --> (Перегляд історії)

    Бібліотекар --> (Управління каталогом)
    Бібліотекар --> (Видача/Прийом книги)
    Бібліотекар --> (Генерація звітів)

    Адміністратор --> (Керування користувачами)
```
